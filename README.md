# WPF Hangman Game

This project is a Windows desktop game built with C# and WPF, implementing the classic Hangman game. The application features user management, game persistence, statistics tracking, and a structured MVVM architecture.

## Features

- User management (create, select, delete users)
- Hangman gameplay with categories and timer
- Save and load game progress
- Statistics per user
- Menu navigation and keyboard shortcuts

## Technologies

- C#
- WPF (.NET Core)
- MVVM architecture

## Purpose

This project was created as a learning exercise to explore WPF desktop development, focusing on building an interactive UI, applying MVVM principles, and implementing core game features.
