# WPF Hangman Game

This project is a Windows desktop game built with C# and WPF, implementing the classic Hangman game. The project focuses on separation of concerns, user interaction, and data persistence while providing a fully functional and interactive gameplay experience.

## Features

- User management (create, select, delete profiles)
- Hangman gameplay with multiple categories and timer
- Save and load game progress
- User-specific statistics tracking
- Keyboard input support for letter guessing
- Menu navigation and commands

## Technologies

- C#
- WPF (.NET Core)
- MVVM architecture
- JSON-based persistence

## Purpose

This project was created as a learning exercise to explore WPF desktop development, focusing on:
- building a responsive UI
- applying MVVM principles
- implementing clean architecture practices
- managing application state and persistence
